# 2026-06-11: ADR-002 — Goal Amped: AFK First, Lenny Earns Its PRD

## Summary

The roadmap's headline ambition was raised and its sequencing inverted, recorded as
[ADR-002](.orchestra/adr/ADR-002-afk-first.md). The objective is no longer "a suite of SDLC
tools" with AFK as the final milestone — it is **one Orchestra-managed work item going from
intake to shipped, AFK, with humans only at the PRD and spec gates**. The milestone order
becomes M5 → M3 → M4: prove the AFK loop with the cheapest loop-owner first, and let what
breaks design the conductor.

## The Decision

Three candidate framings were weighed at the gate (human picked the first):

1. **AFK-first** (chosen) — headless Claude Code, which exists today and already speaks MCP,
   becomes the first scheduled loop-owner. The orchestra-pilot cold-agent test runs first to
   measure whether the deployed tool catalog triggers skills unprompted.
2. **Lenny maximal** — merge M3+M5 and build the Durable Object conductor with scheduling and
   async gates from day one. Rejected: designs the largest build from speculation, no
   intermediate proof point.
3. **Self-hosting first** — the AFK pipeline builds M4 Workshop as its opening act. Rejected as
   the starting point (debugging the loop while delivering real scope couples two failure
   domains) but kept as the finish line: M4 should ideally ship *by* the pipeline.

The pivotal inversion: **M5 no longer depends on M3 — M3's PRD is an output of M5's first
runs.** The skills-as-a-service devlog (2026-06-10) already established that the MCP layer is
harness-agnostic and that headless Claude Code is the cheapest first AFK run; ADR-002 turns
that observation into the score. No Lenny implementation starts before at least one scheduled
headless run has produced a failure list.

## Guardrails Kept

Human gates are unchanged and non-negotiable: AFK means away from keyboard, not human-free.
Gate notifications must reach a human and block until answered — that requirement moved up
into the roadmap's success criteria rather than living only inside M5.
