# M3 — Polish & Distribution

**Objective:** Make `relnotes` pleasant to install and use in the wild — packaged
on PyPI (`pipx install relnotes`), robust error messages, helpful `--help` with
examples, and tuned heuristics for repos that don't use Conventional Commits.

## Success Criteria

- [ ] `pipx install relnotes` yields a working `relnotes` command.
- [ ] `--help` documents every flag with realistic examples.
- [ ] Graceful, actionable errors for: not a git repo, missing `git`, unknown ref,
      empty range, dirty/no tags.
- [ ] Improved fallback heuristics for non-CC repos, validated on ≥2 real public
      repositories.
- [ ] README usage section and a CHANGELOG generated by relnotes itself (dogfood).

## Context

Final milestone of the [relnotes Roadmap](../../roadmap.md). Turns the working
engine into a distributable, polished tool.
